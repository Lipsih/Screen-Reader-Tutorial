Screen Reader Tutorial

Welcome follow the instructions below.

Setup

This tutorial is for these specifications:

  • Windows 10
  • Chrome Version 110.0.5481.78
  • Windows 10 Narrator

How to Operate Screen Reader

Open and close the screen reader with Win + Ctrl + ENTER

Test the Screen Reader

A website that functions well with this Screen Reader is H&M's Website

A website that functions less than optimal with this Screen Reader is Taste of India's Site

Useful Hotkeys

HotkeyDescriptionExample
Win + the Plus sign Keybind to speed up narrator voice You want to speed up the voice
Win + the Minus sign Keybind to slow down narrator voice You want to slow down the voice
Home Move to and read the first character of a line You want to move to the start of a line
End Move to and read the last character of a line You want to jump to the end of a line
Ctrl + Down arrow Read next line Jump to the next line
Ctrl + Up arrow Read previous line Jump to previous
Down arrow Jump to next text or item You want to go the next text of item
Up arrow Jump to previous text or item Example
Caps Lock + X Jump to previous text or item Example
