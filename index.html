Screen Reader Tutorial

Welcome follow the instructions below.

Setup

This tutorial is for these specifications:

  • Windows 10
  • Chrome Version 110.0.5481.78
  • Windows 10 Narrator

How to Operate Screen Reader

Open and close the screen reader with Win + Ctrl + ENTER

Test the Screen Reader

A website that functions well with this Screen Reader is H&M's Website

, as it provides easy access to the navigation at the top, and is also rated as of the most accessible websites out there

A website that functions less than optimal with this Screen Reader is Taste of India's Site

, just try and even move around the screen with arrow keys, and it won't let you move around

Useful Hotkeys

HotkeyDescriptionExample
Win + the Plus sign Keybind to speed up narrator voice You can use this on H&M's site to speed up the speed at which the narrator speaks
Win + the Minus sign Keybind to slow down narrator voice You can use this on H&M's site to speed up the speed at which the narrator speaks
Home Move to and read the first character of a line You can use this on H&M's site to reach the beginning of the current line
End Move to and read the last character of a line You can use this on H&M's site to reach the ending of the current line
Ctrl + Down arrow Read next line You can use this on H&M's site to reach the next line in a paragraph or other element
Ctrl + Up arrow Read previous line You can use this on H&M's site to reach the last line in a paragraph or other element
Down arrow Jump to next text or item You can use this on H&M's site to reach the next html item in the page
Up arrow Jump to previous text or item You can use this on H&M's site to reach the previous html item in the page
Caps Lock + X Repeat Previous narrator item You can use this on H&M's site to repeat anything the narrator says that you may have quickly missed and need to hear again
ENTER Used to click links and other interactive elements on a page You can use this on H&M's site to click any links and interact with buttons and other items on the site
